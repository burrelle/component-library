# Avatar

### Usage

```jsx
import { Avatar } from '@burrelle/flight';
```

<!-- STORY -->

- `name` - `user name.`
- `color` - `background color choice of a user.`
- `height` - `height`
- `width` - `width`

| propName | propType |
| -------- | :------: |
| `name`   | `string` |
| `color`  | `string` |
| `height` | `number` |
| `width`  | `number` |

### Example

```jsx
import { Avatar } from '@burrelle/flight';

render() {
  return <Avatar name={name} color={color} height={height} width={width} />;
}
```
